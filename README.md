### Karaoke Song Finder - Android App (v1.2.0 - 2026/06/28)

Android App built using Kotlin that lets you search karaoke songs from the TJ Karaoke library.


### Main features:
- Ability to search Karaoke songs for the TJ Karaoke system by song name or artist name.
- Ability to look up latest songs added to the TJ Karaoke system, and search by year/month a song was added.
- Ability to add or remove any searched songs and view them on the favourites page.
- View additional song details such as release date, composers and lyricists
